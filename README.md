# oruizrui.github.io

Personal website built with Jekyll and the [Minima theme](https://jekyll.github.io/minima/).

## Development

### Docker (Recommended)

```bash
docker compose up
```

### Local

```bash
bundle install
bundle exec jekyll serve
```

Site available at http://localhost:4000

## Deployment

Push to `main` branch to deploy via GitHub Pages.
